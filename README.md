# Lap Ring Journey with Credit Pulse

A React-based web application for the Lap Ring Journey, featuring credit pulse indicators and responsive layouts for both mobile and desktop.

## Project Structure

- `lap-journey/`: Main React application (Vite + TypeScript + Tailwind CSS)
- `docs/`: Design specifications and planning documents (located within `lap-journey/docs`)

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd lap-journey
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Responsive Design**: Auto-detects mobile or desktop and provides optimized layouts.
- **Credit Pulse components**: Score gauges, credit factors, and journey trackers.
- **Three Journey Paths**:
  - Good Score (6 screens)
  - Poor Score (8 screens)
  - Almost There (7 screens)

## Technologies Used

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
