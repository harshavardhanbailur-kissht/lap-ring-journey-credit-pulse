# Lap Ring Journey with Credit Pulse

A React-based web application for the Lap Ring Journey, featuring credit pulse indicators and responsive layouts for both mobile and desktop.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Responsive Design**: Auto-detects mobile or desktop and provides optimized layouts.
- **Credit Pulse components**: Score gauges, credit factors, and journey trackers.
- **Three Journey Paths**:
  - Good Score (6 screens)
  - Poor Score (8 screens)
  - Almost There (7 screens)

## Technologies Used

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
